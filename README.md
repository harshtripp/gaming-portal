# Mario Game Website

This project is a simple web-based Mario game embedded inside a personal website. Users can play the game directly on the homepage.

---

## Features

* Fully playable Mario game embedded on the website.
* Responsive design for desktop and mobile.
* Styled homepage with custom CSS.

---

## Project Structure

```
/mario-website
│
├── index.html       # Main HTML file
├── style.css        # Stylesheet for the website
├── mario-game.js    # (Optional) JavaScript for Mario game
└── assets/          # Folder for images, sprites, or sounds
```

---

## How to Run

1. Clone or download the repository.
2. Open `index.html` in a web browser (Chrome/Firefox recommended).
3. Play the embedded Mario game directly on the homepage.

---

## Deployment

You can host this website on any static web hosting service such as:

* GitHub Pages
* Netlify
* Vercel

---

## Notes

* Ensure all asset paths in `index.html` are correct.
* For best performance, use a modern browser.
* This is a client-side only project; no backend is required.
